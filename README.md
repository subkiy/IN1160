# IN1160
An introduction in machine learning held in University of Oslo. 

Course Overview: Introduction to Machine Learning and AI
This course provides a fundamental introduction to Machine Learning (ML) and Artificial Intelligence (AI). The curriculum focuses on practical applications—such as language technology, image processing, and robotics—and aims to provide students with a conscious approach to methodology and the execution of ML experiments using Python programming.

The course covers:
- Supervised Learning (for both regression and classification)
- Unsupervised Learning (clustering)
- Reinforcement Learning
- Experimental Design and Evaluation

Additionally, students will receive an introduction to fundamental philosophical issues and ethical questions related to ML/AI, as well as the history of the field.
